# Search

The search feature lets you find tokens and traders quickly from anywhere in the app.

## How to Search

Tap the **search icon** in the header to open the search modal. You can also access search from the dedicated Search page.

## Finding Tokens

Search for tokens by:

- **Name** — e.g., "Wrapped CRO"
- **Symbol** — e.g., "WCRO"
- **Contract address** — paste the full token address

Results show the token name, symbol, price, and liquidity. Tap a result to go directly to its trade page.

## Finding Traders

Search for wallets by:

- **Wallet address** — paste a full `0x...` address
- **.cro name** — type a Cronos ID like `alice.cro`
- **Username** — search by registered display name

Tap a trader result to view their profile page.

## Search Results

Results are ranked by relevance and liquidity. Tokens with higher trading volume and liquidity appear first.

The search updates as you type, showing results in real-time without needing to press enter.
